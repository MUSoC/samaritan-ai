# Sentiment Classifier

## Command sequence to run
```shell
python stem.py
python tokens.py
python sentiment.py
python classify.py
```

## What it does
* Uses PorterStemmer to find stems of all tokenize
* Tokenize input data set
* Build Logistic Regression model, evaluate, save and use it
